flex-direction : 정렬할 방향 지정
row -> 왼쪽에서 오른쪽
row-reverse -> 오른쪾에서 왼쪽
col -> 위에서 아래
col-reverse -> 아래에서 위


flex-wrap : flex 요소들을 한 줄 또는 여러 줄에 걸쳐 정렬
nowrap(기본) -> 아이템이 한 줄에 붙게 됨.
wrap -> 한 줄에 꽉 차게 되면, 다음 라인으로 넘어가게 됨



flex-flow : flex-direction + flex-wrap을 합침
ex )
flex-flow : column nowrap <- 위에서 아래로


justify-content : flex 요소들을 flex-direction에서 정한 방향 상에서 정렬
속성
flex-start : 왼쪽에서 시작
flex-end : 오른쪽에서 시작
* justify-content와 flex-wrap의 차이는, 순서에 영향을 줌
space-around :
space-evenly : 같은 간격
space-between : 왼쪽과 오른쪽 끝은 끝에 붙어 있고, 나머지 간격 유지


align-items : flex 요소를 flex-direction에서 정한 방향 수직 상에서 정함
속성
baseline : text가 균등하게 배치됨

align-content : 세로선 상에 여분의 공간이 있는 경우 Flex 컨테이너 사이의 간격 조절
속성
space-between :위 아래는 붙어있고, 중간에 아이템이 들어감
center : 중간에 모임


flex-grow : 사이트 사이즈가 커지면 container를 꽉 채우려고 유지함

flex-basis : 공간을 얼마나 차지해야 하는지
속성
auto


align-self : 아이템 별로 아이템 정렬 가능
